Introduction

Regarding this version, Hosket is still in development so there are not many programming features yet. In this version, it is simply a programming language with simple programs, command-line programs, etc.

Hosket's syntax is related and closely linked to command-line languages. The programming language is written in a syntax that may differ between languages, creating a difference and a special name for Hosket.

Hosket is not only special in syntax or features, but it is a programming language created by a Vietnamese. In Vietnam, when it comes to programming, most people focus on Web Development and Data Science, but I started sitting at the computer desk to write a perfect programming language!

Hosket is also a completely free software when you do not need to spend a money for this software.
